Este projeto segue as boas práticas de gestão de projetos orientadas pelo PMBOK, cobrindo todo o ciclo de vida dos dados, desde a concepção do problema até a geração de valor por meio de análises, dashboards, modelos de Machine Learning e Data Apps.

O planejamento foi estruturado em etapas sequenciais e interdependentes, garantindo governança, qualidade dos dados e escalabilidade da solução, alinhadas ao ciclo de vida de dados adotado pela Dadosfera.
